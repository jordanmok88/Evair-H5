# Evair H5

H5 web app based on Evair APP (Evair-APP-Cursor-Code).

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
